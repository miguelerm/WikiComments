WikiComments
============

Extension para MediaWiki que permite agregar comentarios a un articulo cuando
se agrega la funcion {{#commentsform:}} dentro del texto del mismo.

Adicionalmente permite mostrar la lista de comentarios del artículo si se agrega
la funcion {{#commentslist:}} dentro del texto del mismo.

Estado del proyecto
-------------------

Actualmente el proyecto se encuentra en fase de desarrollo, no es aconsejable
instalarlo.
